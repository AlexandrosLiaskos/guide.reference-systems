# Mendeley Reference Manager Guide

## Overview

Mendeley is a free reference manager that helps you organize research, collaborate, and generate citations and bibliographies.

---

## Getting Started

### Installation
1. Download from [mendeley.com](https://www.mendeley.com)
2. Create a free account
3. Install desktop app and browser extension

### Components
- **Mendeley Reference Manager** - Desktop/web app for organizing
- **Web Importer** - Browser extension to save sources
- **MS Word Plugin** - Insert citations while writing
- **Mobile App** - Access library on the go

---

## Adding References

### From Browser (Web Importer)
1. Install the Mendeley Web Importer browser extension
2. Navigate to any article, book, or webpage
3. Click the Mendeley icon in browser toolbar
4. Select the library/folder to save to
5. Click "Save"

### From PDF
1. Drag and drop PDF into Mendeley
2. Mendeley extracts metadata automatically
3. Verify/edit details if needed

### Manual Entry
1. Click "Add" → "Add Entry Manually"
2. Select document type (Journal Article, Book, etc.)
3. Fill in required fields:
   - Authors
   - Title
   - Year
   - Journal/Publisher
   - Volume, Issue, Pages
   - DOI/URL

### Import from Database
1. Search in Google Scholar, PubMed, etc.
2. Export citation as .RIS, .BibTeX, or .EndNote
3. In Mendeley: File → Import → Select file

### By DOI/PMID/ArXiv ID
1. Click "Add" → "Add Entry Manually"
2. Paste DOI, PMID, or ArXiv ID
3. Click search icon - metadata auto-fills

## Organizing Your Library

### Creating Folders
1. Click "New Folder" in left sidebar
2. Name your folder (e.g., "Thesis Chapter 1")
3. Drag references into folders
4. References can exist in multiple folders

### Using Tags
1. Select a reference
2. In details panel, find "Tags" field
3. Add keywords (e.g., "methodology", "review")
4. Filter library by tags

### Starring Important References
- Click the star icon on any reference
- Filter to show only starred items

### Search & Filter
- **Search box**: Search titles, authors, keywords
- **Filter by**: Author, Tag, Publication, Year
- **Sort by**: Author, Title, Year, Date Added

---

## Managing PDFs

### Attaching PDFs
- Drag PDF onto existing reference
- Or import PDF directly (metadata extracted)

### Renaming PDFs
1. Go to Tools → Options → File Organizer
2. Set naming pattern: `Author - Year - Title.pdf`
3. Enable "Organize my files"

### Annotating PDFs
1. Double-click reference to open PDF
2. Use toolbar for:
   - Highlight text (multiple colors)
   - Add sticky notes
   - Draw/underline
3. Annotations sync across devices

## Using with Microsoft Word

### Installing the Plugin
1. In Mendeley: Tools → Install MS Word Plugin
2. Restart Word
3. "Mendeley Cite" tab appears in Word ribbon

### Inserting Citations
1. Place cursor where citation should go
2. Click "Insert Citation" in Mendeley tab
3. Search for reference by author/title/year
4. Select reference(s) and click "Insert"

### Inserting Multiple Sources
1. Click "Insert Citation"
2. Search and select first reference
3. Click "+" to add more references
4. Click "Insert" when done

### Adding Page Numbers
1. Insert citation normally
2. Click on the citation in your document
3. Click "Edit Citation" in Mendeley tab
4. Add page number in "Page" field
5. Click "Save"

### Generating Bibliography
1. Place cursor where bibliography should appear
2. Click "Insert Bibliography" in Mendeley tab
3. Bibliography generates automatically
4. Updates when you add/remove citations

### Changing Citation Style
1. Click "Citation Style" dropdown in Mendeley tab
2. Select from common styles or "More Styles"
3. Search for any style (2000+ available)
4. All citations and bibliography update automatically

## Collaboration Features

### Sharing References
1. Create a Group: Click "New Group"
2. Name the group and set privacy
3. Invite members by email
4. Shared references appear in group folder

### Group Types
- **Private**: Invite-only, hidden from public
- **Invite-only**: Listed publicly, join by invitation
- **Public**: Anyone can join and view

---

## Syncing Across Devices

### Automatic Sync
- References sync automatically when online
- Click sync icon to force sync
- 2GB free cloud storage for PDFs

### Web Library
- Access at mendeley.com/reference-manager
- Full library available in browser
- Add/edit references from any computer

---

## Exporting References

### Export Selected References
1. Select references (Ctrl/Cmd + click for multiple)
2. File → Export
3. Choose format:
   - BibTeX (.bib)
   - RIS (.ris)
   - EndNote XML

### Export Entire Library
1. Select all (Ctrl/Cmd + A)
2. File → Export
3. Choose format and location

## Troubleshooting

### Citation Not Appearing in Word
- Ensure Word plugin is installed (Tools → Install MS Word Plugin)
- Restart Word after installation
- Check Mendeley desktop is running

### PDF Metadata Wrong
1. Select the reference
2. Click "Check for Duplicates" or manually edit
3. Use "Search by DOI" to find correct metadata

### Duplicates in Library
1. Tools → Check for Duplicates
2. Review suggested duplicates
3. Click "Merge" to combine entries

### Sync Issues
1. Click sync icon to force sync
2. Check internet connection
3. Log out and back in to Mendeley account
