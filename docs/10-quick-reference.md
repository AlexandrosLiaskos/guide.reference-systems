# Quick Reference Charts

## Inline Citation Comparison

| Style | Format | Example |
|-------|--------|---------|
| **APA** | (Author, Year) | (Smith, 2024) |
| **MLA** | (Author Page) | (Smith 45) |
| **Chicago (Author-Date)** | (Author Year, Page) | (Smith 2024, 45) |
| **Chicago (Notes)** | Superscript → Footnote | ¹ → Footnote text |
| **Harvard** | (Author Year, p. #) | (Smith 2024, p. 45) |
| **IEEE** | [#] | [1] |
| **Vancouver** | (#) or superscript | (1) or ¹ |
| **AMA** | Superscript | ¹ |
| **Bluebook** | Footnote | ¹ → Full legal citation |

## Bibliography Entry: Book (Single Author)

### APA
```
Smith, J. D. (2024). Title of book: Subtitle. Publisher. https://doi.org/xxxxx
```

### MLA
```
Smith, John David. Title of Book: Subtitle. Publisher, 2024.
```

### Chicago (Bibliography)
```
Smith, John David. Title of Book: Subtitle. Place: Publisher, 2024.
```

### Harvard
```
Smith, J.D. (2024) Title of book: subtitle. Place: Publisher.
```

### IEEE
```
[1] J. D. Smith, Title of Book: Subtitle. City, State, USA: Publisher, 2024.
```

### Vancouver
```
1. Smith JD. Title of book: subtitle. Place: Publisher; 2024.
```

### AMA
```
1. Smith JD. Title of Book: Subtitle. Publisher; 2024.
```

## Bibliography Entry: Journal Article

### APA
```
Smith, J. D. (2024). Title of article. Journal Name, 45(3), 234-251. https://doi.org/xxxxx
```

### MLA
```
Smith, John David. "Title of Article." Journal Name, vol. 45, no. 3, 2024, pp. 234-51.
```

### Chicago (Bibliography)
```
Smith, John David. "Title of Article." Journal Name 45, no. 3 (2024): 234-51.
```

### Harvard
```
Smith, J.D. (2024) 'Title of article', Journal Name, 45(3), pp. 234-251. doi: xxxxx
```

### IEEE
```
[1] J. D. Smith, "Title of article," Journal Name, vol. 45, no. 3, pp. 234-251, Mar. 2024.
```

### Vancouver
```
1. Smith JD. Title of article. Journal Name. 2024;45(3):234-51.
```

### AMA
```
1. Smith JD. Title of article. Journal Name. 2024;45(3):234-251.
```

## Bibliography Entry: Website

### APA
```
Smith, J. (2024, March 15). Title of page. Website Name. https://www.example.com
```

### MLA
```
Smith, John. "Title of Page." Website Name, 15 Mar. 2024, www.example.com.
```

### Chicago (Bibliography)
```
Smith, John. "Title of Page." Website Name. March 15, 2024. https://www.example.com.
```

### Harvard
```
Smith, J. (2024) Title of page. Available at: https://www.example.com (Accessed: 20 March 2024).
```

### IEEE
```
[1] J. Smith. "Title of page." Website Name. https://www.example.com (accessed Mar. 20, 2024).
```

### Vancouver
```
1. Smith J. Title of page [Internet]. Place: Publisher; 2024 [cited 2024 Mar 20]. Available from: https://www.example.com
```

### AMA
```
1. Smith J. Title of page. Website Name. Published March 15, 2024. Accessed March 20, 2024. https://www.example.com
```

## Quick Decision Guide

**Need page numbers in inline citation?**
- Yes: MLA, Chicago Notes
- Only for quotes: APA, Harvard, Chicago Author-Date
- No (uses reference number): IEEE, Vancouver, AMA

**Uses footnotes/endnotes?**
- Yes: Chicago Notes-Bibliography, Bluebook
- No: APA, MLA, Harvard, IEEE, Vancouver, AMA

**Alphabetical or numerical reference list?**
- Alphabetical: APA, MLA, Chicago, Harvard
- Numerical: IEEE, Vancouver, AMA
