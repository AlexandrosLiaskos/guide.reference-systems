# Bluebook Style (21st Edition)

Legal citation format used in US courts, law reviews, and legal documents.

## Inline Citation Format

**Uses footnotes with full citations.** Also has "short form" for subsequent references.

| Citation Type | Usage |
|--------------|-------|
| Full citation | First time citing a source |
| Short form | Subsequent citations to same source |
| Id. | Citing the immediately preceding source |
| Supra | Referring to a source cited earlier |

---

## Cases

### Format - Full Citation:
```
Party v. Party, Volume Reporter Page (Court Year).
```
**Example - Supreme Court:**
```
Brown v. Board of Education, 347 U.S. 483 (1954).
```

**Example - Circuit Court:**
```
Smith v. Jones, 123 F.3d 456 (9th Cir. 2024).
```

**Example - District Court:**
```
Doe v. Roe, 456 F. Supp. 3d 789 (S.D.N.Y. 2024).
```

**Example - State Court:**
```
People v. Smith, 123 Cal. App. 4th 456 (2024).
```

---

### Short Forms:
```
Brown, 347 U.S. at 485.
Smith, 123 F.3d at 458.
Id. at 490.
```

---

### Unpublished/Slip Opinion:
```
Party v. Party, No. 24-1234, slip op. at 5 (Court Month Day, Year).
```
**Example:**
```
Doe v. Roe, No. 24-1234, slip op. at 5 (S.D.N.Y. Mar. 15, 2024).
```

---

## Statutes

### Format - Federal Statute:
```
Title U.S.C. § Section (Year).
```
**Example:**
```
42 U.S.C. § 1983 (2018).
```

---

### Format - Named Act:
```
Name of Act § Section, Volume Stat. Page (Year) (codified at Title U.S.C. § Section).
```
**Example:**
```
Americans with Disabilities Act of 1990 § 102, 104 Stat. 331 (codified at 42 U.S.C. § 12112).
```

---

### Format - State Statute:
```
State Code Ann. § Section (Year).
```
**Example:**
```
Cal. Civ. Code § 1542 (West 2024).
N.Y. Gen. Bus. Law § 349 (McKinney 2024).
```

---

## Constitutions

### Format - U.S. Constitution:
```
U.S. Const. art. #, § #.
U.S. Const. amend. #.
```
**Examples:**
```
U.S. Const. art. I, § 8, cl. 3.
U.S. Const. amend. XIV, § 1.
```

---

### Format - State Constitution:
```
State Const. art. #, § #.
```
**Example:**
```
Cal. Const. art. I, § 7.
```

---

## Regulations

### Format - Code of Federal Regulations:
```
Title C.F.R. § Section (Year).
```
**Example:**
```
17 C.F.R. § 240.10b-5 (2024).
```

---

### Format - Federal Register:
```
Title Fed. Reg. Page (Month Day, Year) (to be codified at Title C.F.R. pt. #).
```
**Example:**
```
89 Fed. Reg. 12345 (Mar. 15, 2024) (to be codified at 40 C.F.R. pt. 60).
```

---

## Legislative Materials

### Format - Bill:
```
Bill Name, Bill No., Congress (Year).
```
**Example:**
```
Clean Energy Act of 2024, H.R. 1234, 118th Cong. (2024).
S. 5678, 118th Cong. (2024).
```

---

### Format - Hearing:
```
Title: Hearing Before the Subcommittee, Committee Cong. Page (Year) (statement of Witness).
```
**Example:**
```
Climate Policy: Hearing Before the S. Comm. on Environment, 118th Cong. 45 (2024) (statement of Dr. Jane Smith, Professor, Harvard University).
```

---

### Format - Congressional Report:
```
House/Senate Rep. No. #-#, at Page (Year).
```
**Example:**
```
H.R. Rep. No. 118-123, at 15 (2024).
S. Rep. No. 118-456, at 22 (2024).
```

---

### Format - Congressional Record:
```
Volume Cong. Rec. Page (Year).
```
**Example:**
```
170 Cong. Rec. S1234 (daily ed. Mar. 15, 2024).
```

---

## Books & Treatises

### Format - Single Author:
```
Author Full Name, Title of Book Page (Edition Year).
```
**Example:**
```
John D. Smith, The Complete Guide to Legal Research 45 (4th ed. 2024).
```

---

### Format - Multiple Authors:
```
Author One & Author Two, Title of Book Page (Year).
```
**Example:**
```
Mary Johnson & Kevin Williams, Legal Methods Explained 123 (2024).
```

---

### Format - Editor:
```
Title of Book Page (Editor ed., Year).
```
**Example:**
```
Handbook of Legal Practice 89 (Robert Martinez ed., 3rd ed. 2024).
```

---

### Format - Chapter in Book:
```
Author, Chapter Title, in Title of Book Page (Editor ed., Year).
```
**Example:**
```
Sarah Thompson, Evidence Analysis, in Handbook of Legal Practice 123 (Robert Martinez ed., 2024).
```

---

## Law Review Articles

### Format - Print:
```
Author, Article Title, Volume Journal Abbreviation Page (Year).
```
**Example:**
```
John Smith, Constitutional Interpretation in the Modern Era, 123 Harv. L. Rev. 456 (2024).
```

---

### Format - Online:
```
Author, Article Title, Volume Journal Abbreviation Page (Year), URL.
```
**Example:**
```
Mary Johnson, Digital Privacy Rights, 45 Yale L.J. Online 12 (2024), https://www.yalelawjournal.org/example.
```

---

### Format - Student Work (Note/Comment):
```
Author, Note/Comment, Article Title, Volume Journal Abbreviation Page (Year).
```
**Example:**
```
Jane Doe, Note, Corporate Liability Standards, 98 N.Y.U. L. Rev. 789 (2024).
```

---

## Periodicals & Newspapers

### Format - Newspaper:
```
Author, Article Title, Newspaper Name, Month Day, Year, at Page.
```
**Example:**
```
Adam Liptak, Supreme Court Rules on Privacy, N.Y. Times, Mar. 20, 2024, at A1.
```

---

### Format - Magazine:
```
Author, Article Title, Magazine Name, Month Day, Year, at Page.
```
**Example:**
```
John Roberts, The Future of the Court, Time, Apr. 1, 2024, at 42.
```

---

## Internet Sources

### Format - Website:
```
Author, Title of Page, Website Name (Month Day, Year), URL.
```
**Example:**
```
Legal Information Institute, Fourth Amendment, Cornell Law School (Mar. 15, 2024), https://www.law.cornell.edu/constitution/fourth_amendment.
```

---

### Format - Blog Post:
```
Author, Post Title, Blog Name (Month Day, Year), URL.
```
**Example:**
```
Eugene Volokh, Free Speech Update, Volokh Conspiracy (Mar. 15, 2024), https://reason.com/volokh/example.
```

---

## International Materials

### Format - Treaty:
```
Treaty Name, Parties, Date, Volume Treaty Source Page.
```
**Example:**
```
Vienna Convention on the Law of Treaties, May 23, 1969, 1155 U.N.T.S. 331.
```

---

### Format - U.N. Document:
```
U.N. Body, Title, U.N. Doc. Number (Year).
```
**Example:**
```
U.N. Human Rights Council, Report on Digital Rights, U.N. Doc. A/HRC/45/22 (2024).
```

---

### Format - Foreign Case:
```
Case Name, [Year] Volume Reporter Page (Court) (Country).
```
**Example:**
```
Regina v. Smith, [2024] 1 W.L.R. 123 (H.L.) (U.K.).
```

## Short Forms Reference

| Full Form | Short Form |
|-----------|------------|
| First citation | Full citation |
| Immediately following | Id. |
| Immediately following, different page | Id. at [page]. |
| Previously cited | [Author], supra note [#], at [page]. |
| Case previously cited | [Short case name], [volume] [reporter] at [page]. |

## Signal Words

| Signal | Meaning |
|--------|---------|
| [no signal] | Directly supports proposition |
| See | Clearly supports |
| See also | Additional support |
| Cf. | Analogous support |
| Compare...with | Comparison illustrates |
| Contra | Directly contradicts |
| But see | Contradicts |
| See generally | Background material |
