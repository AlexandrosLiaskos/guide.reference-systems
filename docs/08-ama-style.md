# AMA Style (11th Edition)

American Medical Association style, used by JAMA and AMA journals.

## Inline Citation Format

**Uses superscript numbers** after punctuation.

| Scenario | Format | Example Text |
|----------|--------|--------------|
| Single source | ˣ | ...significant results.¹ |
| Multiple sequential | ˣ⁻ˣ | ...multiple studies.¹⁻³ |
| Multiple non-sequential | ˣ,ˣ | ...research shows.¹,⁴,⁷ |

**Rules:**
- Superscript numbers placed after punctuation (periods, commas)
- Numbers assigned in order of first appearance
- Same source uses same number throughout
- References listed numerically

---

## Reference List Format & Examples

**Title:** "References"
**Order:** Numerical (order of citation)
**Author format:** Up to 6 authors list all; 7+ list first 3 then "et al."

---

### Books

**Format - Single Author:**
```
#. Author AA. Title of Book: Subtitle. Publisher; Year.
```
**Example:**
```
1. Smith JD. The Complete Guide to Clinical Research: Methods and Practice. Academic Press; 2024.
```

---

**Format - Two to Six Authors:**
```
#. Author AA, Author BB, Author CC. Title of Book. Publisher; Year.
```
**Example:**
```
2. Johnson MR, Williams KL. Research Methods in Medicine. Sage Publications; 2023.
```

---

**Format - Seven+ Authors:**
```
#. Author AA, Author BB, Author CC, et al. Title of Book. Publisher; Year.
```
**Example:**
```
3. Adams A, Brown B, Chen C, et al. Collaborative Medical Research. Oxford University Press; 2024.
```

---

**Format - Edited Book:**
```
#. Editor AA, ed. Title of Book. Publisher; Year.
```
**Example:**
```
4. Martinez R, ed. Handbook of Clinical Methods. Routledge; 2024.
```

---

**Format - Chapter in Edited Book:**
```
#. Author AA. Title of chapter. In: Editor BB, ed. Title of Book. Publisher; Year:pages.
```
**Example:**
```
5. Thompson S. Data analysis in clinical trials. In: Martinez R, ed. Handbook of Clinical Methods. Routledge; 2024:123-156.
```

---

**Format - Edition Other Than First:**
```
#. Author AA. Title of Book. #th ed. Publisher; Year.
```
**Example:**
```
6. Smith JD. Medical Research Basics. 4th ed. Academic Press; 2024.
```

---

### Journal Articles

**Format - Standard Article (Print):**
```
#. Author AA, Author BB. Title of article. Journal Name Abbrev. Year;Volume(Issue):Pages.
```
**Example:**
```
7. Williams KL, Brown MA. Effects of treatment methodology. JAMA. 2024;331(12):1034-1041.
```

---

**Format - With DOI:**
```
#. Author AA, Author BB. Title of article. Journal Name Abbrev. Year;Volume(Issue):Pages. doi:DOI
```
**Example:**
```
8. Brown MA, Chen RS. Digital interventions in healthcare. Ann Intern Med. 2023;178(4):523-531. doi:10.7326/example
```

---

**Format - Online Only:**
```
#. Author AA. Title of article. Journal Name Abbrev. Published online Month Day, Year. doi:DOI
```
**Example:**
```
9. Chen RS. Innovative treatment approaches. JAMA Netw Open. Published online March 15, 2024. doi:10.1001/example
```

---

**Format - Epub Ahead of Print:**
```
#. Author AA. Title of article. Journal Name Abbrev. Published online Month Day, Year. doi:DOI
```
**Example:**
```
10. Davis LM. Emerging therapies. N Engl J Med. Published online March 10, 2024. doi:10.1056/example
```

---

**Format - Article with Article Number:**
```
#. Author AA. Title of article. Journal Name Abbrev. Year;Volume(Issue):article number. doi:DOI
```
**Example:**
```
11. Williams RT. Neural correlates study. Nature. 2024;607(7917):e12345. doi:10.1038/example
```

---

### Websites

**Format - With Author:**
```
#. Author AA. Title of page. Website Name. Published Month Day, Year. Accessed Month Day, Year. URL
```
**Example:**
```
12. Johnson T. Understanding clinical trials. ClinicalTrials.gov. Published March 15, 2024. Accessed March 20, 2024. https://clinicaltrials.gov/methods
```

---

**Format - Organization Author:**
```
#. Organization. Title of page. Website Name. Published Month Day, Year. Accessed Month Day, Year. URL
```
**Example:**
```
13. American Medical Association. Publication guidelines. AMA Manual of Style. Accessed March 20, 2024. https://www.amamanualofstyle.com/guidelines
```

---

**Format - No Date:**
```
#. Author AA. Title of page. Website Name. Accessed Month Day, Year. URL
```
**Example:**
```
14. Smith R. Treatment protocols. Medical Resource. Accessed March 20, 2024. https://www.example.com/protocols
```

**Example:**
```

```

---

**Format - Newspaper (Online):**
```
#. Author AA. Title of article. Newspaper Name. Month Day, Year. Accessed Month Day, Year. URL
```
**Example:**
```
16. Johnson L. Study reveals treatment breakthrough. Washington Post. March 20, 2024. Accessed March 21, 2024. https://www.washingtonpost.com/example
```

---

**Format - Magazine:**
```
#. Author AA. Title of article. Magazine Name. Month Year:Pages.
```
**Example:**
```
17. Brown K. The future of medicine. Scientific American. April 2024:42-49.
```

### Reports

**Format - Government Report:**
```
#. Organization. Title of Report. Publisher; Year. Publication No.
```
**Example:**
```
18. National Institutes of Health. Clinical Research Standards 2024. US Government Printing Office; 2024. NIH Publication No. 24-001.
```

---

**Format - Technical Report:**
```
#. Author AA. Title of Report. Organization; Year. Report No.
```
**Example:**
```
19. Smith J, Jones M. Analysis of Treatment Outcomes. Research Institute; 2024. Report TR-2024-15.
```

---

### Conference Papers

**Format - Published Proceedings:**
```
#. Author AA. Title of paper. In: Editor BB, ed. Title of Proceedings. Publisher; Year:Pages.
```
**Example:**
```
20. Johnson K. New approaches to clinical trials. In: Smith J, ed. Proceedings of the Annual Medical Convention. Medical Press; 2024:45-56.
```

---

**Format - Presented Paper:**
```
#. Author AA. Title of paper. Paper presented at: Conference Name; Month Day, Year; City.
```
**Example:**
```
21. Williams R. Methodology innovations. Paper presented at: AMA Annual Meeting; May 15, 2024; Chicago, IL.
```

### Dissertations & Theses

**Format:**
```
#. Author AA. Title of Dissertation. Dissertation. University; Year.
```
**Example:**
```
22. Williams RT. Research Methodology in Clinical Practice. Dissertation. University of Michigan; 2024.
```

### Legal/Legislative Sources

**Format - Law:**
```
#. Title of Law, Pub L No. #-#, Stat # (Year).
```
**Example:**
```
23. Patient Protection and Affordable Care Act, Pub L No. 111-148, 124 Stat 119 (2010).
```

---

**Format - Regulation:**
```
#. Title of regulation, Volume CFR §Section (Year).
```
**Example:**
```
24. Protection of Human Subjects, 45 CFR §46 (2018).
```
